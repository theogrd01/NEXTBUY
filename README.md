# 🛒 Nextbuy

## 📌 Description

Nextbuy is a web application built with Next.js designed to simplify and optimize the online shopping experience.  
The goal of the project is to provide a modern, smooth, and efficient interface for comparing, organizing, and tracking purchases.

## 🎯 Objectives

- Simplify the online purchasing process  
- Provide a clean and intuitive user interface  
- Centralize product information  
- Improve user decision-making  

## 🚀 Key Features

- 🔍 Product search  
- 📊 Product comparison  
- 🧾 Purchase tracking  
- ❤️ Favorites / wishlist management  
- 📱 Responsive design (mobile & desktop)  

## 🛠️ Technologies Used

- Next.js – React framework  
- React – User interface  
- Node.js – Backend  
- CSS / Tailwind / Bootstrap (adapt depending on your project)  
- REST API (if used)  

## 📂 Project Structure

/components     → Reusable components /pages          → Application pages /public         → Static files /styles         → Stylesheets /utils          → Utility functions

## ⚙️ Installation

1. Clone the repository:

bash git clone https://github.com/your-username/nextbuy.git 

2. Navigate into the project folder:

bash cd nextbuy 

3. Install dependencies:

pip install -r requirements.txt

4. Run the development server:

bash npm run dev
